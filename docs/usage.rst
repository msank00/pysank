=====
Usage
=====

To use pysank in a project

- create directory

::

    from pysank.pysank import *
    from pysank import pysank as ps
    import pysank.pysank as ps2

    dirname = 'data_input'
    dir_out = 'data_output'

    ps.create_directory(dirname)
    ps2.create_directory(dir_out)
    create_directory(dirname)
